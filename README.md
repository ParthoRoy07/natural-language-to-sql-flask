# Natural Language to SQL Flask App

A simple web application that converts natural language queries into SQL commands and executes them on a SQLite database using Flask.

---

## 🚀 Features

- Rule-based Natural Language to SQL conversion
- SQLite database integration
- Flask backend
- Dynamic result display
- Simple and clean UI

---

## 🛠 Tech Stack

- Python
- Flask
- SQLite
- Pandas
- HTML / CSS

---

## 📌 Example Queries

- Show all students
- Show students with marks greater than 80
- Show highest marks
- Show CSE students
- Show ECE students

## ▶️ How to Run the Project

1️⃣ Install dependencies:

```
pip install -r requirements.txt
```

2️⃣ Run the application:

```
python app.py
```

3️⃣ Open in browser:

```
http://127.0.0.1:5000
```
